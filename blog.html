Interacting with your own LLM through Python

First, ensure you have a Replit account with enough space for a python app. If it does not have enough space, then create a new account under another email.

Create an App, and select "Python"

Paste the following into your main.py:

import os
        
from groq import Groq
        
client = Groq(
    api_key=os.environ.get("GROQ_API_KEY"),
)
        
chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "What does the federal interest rate do to the economy?",
        }
    ],
    model="llama-3.3-70b-versatile",
)
        
print(chat_completion.choices[0].message.content)

Now, Go to groq.com, and go to Free API Key under Developers

Log in with a google account, then click "Create API Key."

Use any display name.

Copy the display key, and store it somewhere safe.

Now go back to replit, create a new tab in the IDE, and make that tab "Secrets." Add a New Secret, label it "GROQ_API_KEY", and enter the value as your stored key.

Create another tab called "Shell", and type "pip install groq"

Once all packages are downloaded, change the "content" line of your python code to any question you'd like to ask your AI, and run. Your output will appear in the Console tab.

Additional Challenge:

Connect your python file to a flask server so you can alter the JSON message content via a form or other text input, and receive and output in the HTML page.

Or, use the Groq JavaScript library quickstart guide for less dependency
